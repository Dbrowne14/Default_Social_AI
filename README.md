Default Social is an AI-enabled digital media, marketing, and technology platform built with Next.js.

This project focuses on creating a scalable, high-performance web experience that combines modern design, SEO-focused architecture, and future AI integrations for lead generation, content, and user engagement.

## Current Scope

The current MVP build includes:

- Home page
- Services page
- About page
- Insights page
- Responsive navigation and layout
- SEO-friendly structure
- Foundations for CMS and AI integrations

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## Planned Integrations

- Sanity CMS
- AI-assisted content tooling
- Analytics and tracking
- Lead generation workflows
